Tobservable - userDocumentation

  • new tobservable(pData)

    the mainclass, on creation you just put your dataobject in.

  • .get(pPath)

    returns a tobservable that is only can reach the date described by the pPath. this is usefull to get shorter access to values deep in the data-structure.

  • .set(pPath,value,run)

    will set the value given by the part in the data. it run is set of true it will .apply() the function. then the value need to be a array of the arguments.

  • .run(pPath,value)

    shortcut to "set" with the command to run the function.

  • .registerObserver(tObserver, pPath)

    add a new Listener, thet get notified on changes or calls

  • .notify(path)

    If you change the data without the Tobservable, you need to notify the Tobservable about the changes, to keep the listeners on state.
